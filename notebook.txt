Библиотека math.h
    labs()  возвращает модуль числа num, имеющего формат long int
    atof()  преобразует строку str в величину типа double (atof() для строки «100.000HELLO» возвратит 100.00)
    atold() является версией функции atof(), возвращающей значение типа long double
    abs() возвращает модуль целого числа num
    tanh() возвращает гиперболический тангенс аргумента arg
    tan() возвращает тангенс аргумента arg
